# Instagram UI Clone (Figma)

This project is an **Instagram UI/UX clone designed using Figma**.
The goal of this project was to recreate the core user interface of Instagram while practicing **mobile UI design, layout structure, and user experience principles**.

## Features

* Login Page UI
* Home Feed Layout
* Stories Section
* Post View Screen
* Profile Page
* Notifications Page
* Settings Page
* Bottom Navigation Bar

## Tools Used

* Figma (UI/UX Design)
* Design Components
* Auto Layout
* Prototyping

## Project Objective

The objective of this project was to understand how modern social media interfaces are designed and to practice creating clean, responsive, and user-friendly mobile UI layouts.
